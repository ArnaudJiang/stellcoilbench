StellCoilBench
==============

StellCoilBench is an open benchmark suite and AI-automated optimization engine
for stellarator coil design.  It standardizes case definitions and leaderboards
for method comparison, and runs a nonstop autopilot that autonomously explores
the coil design space using a configurable GA or LLM-powered policy.
